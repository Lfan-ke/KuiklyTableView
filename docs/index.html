KuiklyTableView Components

A KuiklyUI table component library for Android, iOS, and HarmonyOS. All demos below are fully interactive - click headers to sort, type to search, expand rows, edit inline, and more.

EditableTableRow - click Edit to switch to inline input fields
EditableTableRow

Click "Edit" on any row to enter inline edit mode, then "Save" to commit changes.

Kotlin DSL
var editingId by observable(-1) var draft by observable<Person?>(null) data.forEach { item -> EditableTableRow( editing = editingId == item.id, onEdit = { editingId = item.id; draft = item.copy() }, onSave = { data = data.map { if (it.id == item.id) draft!! else it }; editingId = -1 }, onCancel = { editingId = -1 } ) { if (editingId == item.id) EditableCell { value(draft!!.name); onChange { draft = draft!!.copy(name = it) } } else TableCell { text(item.name) } } }
